# Northwind SQL Case Study

## Project Overview

This project contains 15 SQL business analysis questions solved using the Northwind dataset.

The project is divided into three levels:

- Easy
- Medium
- Advanced

The objective was to strengthen SQL problem-solving skills and business analytics thinking using real-world business scenarios.

---

## Skills Used

- SQL
- PostgreSQL
- Joins
- Aggregations
- GROUP BY
- ORDER BY
- Subqueries
- Common Table Expressions (CTEs)
- Window Functions
- DENSE_RANK()
- Revenue Analysis
- Contribution Analysis
- Pareto Analysis (80/20 Rule)

---

## Easy Questions

1. Top 3 Categories by Revenue
2. Top 5 Cities by Customer Count
3. Top 5 Products by Revenue
4. Which Year Had the Highest Number of Orders?
5. Which Country Generated the Highest Revenue?

---

## Medium Questions

1. Top 5 Employees by Total Orders Handled
2. Average Order Value (AOV)
3. Top-Selling Product in Each Category
4. Monthly Revenue Trend
5. Top 3 Customers by Revenue

---

## Advanced Questions

1. Product with Highest Revenue Contribution per Category
2. Customer Revenue Contribution Percentage
3. Customers Above Average Revenue
4. Pareto Analysis (80/20 Rule)
5. Product Ranking Within Categories

---

## Dataset

Northwind Database
